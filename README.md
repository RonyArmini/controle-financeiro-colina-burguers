# Sistema de Controle Financeiro - Colina Burguers

Projeto desenvolvido para a disciplina Atividade Extensionista do curso de Análise e Desenvolvimento de Sistemas.

## Tecnologias utilizadas

- Python 3.12
- Tkinter
- SQLite
- Matplotlib

## Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Edição de registros
- Exclusão de registros
- Relatório financeiro
- Visualização gráfica das receitas e despesas
